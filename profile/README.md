## El problema con aprender tecnología

La mayoría de recursos para aprender programación y ciberseguridad comparten el mismo defecto: **te explican todo y te hacen hacer nada**.

Lees. Asientes. Cierras la pestaña. Y al día siguiente no recuerdas nada.

Nosotros lo hacemos al revés.

---

## La filosofía

> **No aprendes a nadar leyendo sobre el agua.**

En Code Intruder aprendes **resolviendo retos reales**. Cada desafío es un problema concreto que tienes que analizar, investigar y descifrar por tu cuenta. Sin pasos guiados. Sin la respuesta al final del capítulo.

Cuando lo consigues, lo recuerdas para siempre.  
Cuando te atascas, aprendes aún más.

## ¿Qué es el formato CTF?

**CTF (Capture The Flag)** es la metodología que usan los mejores profesionales de seguridad para entrenarse. Un reto, una flag oculta, tú y tu cabeza.

Nosotros tomamos ese formato y lo llevamos a todas las áreas del aprendizaje técnico:

|     | Área               | Contenido                             |
| --- | ------------------ | ------------------------------------- |
| 🔐  | **Ciberseguridad** | Web, Forense, Criptografía, Reversing |
| 💻  | **Programación**   | Algoritmos, Lógica, Desarrollo        |
| 🖥️  | **Informática**    | Redes, Sistemas, Fundamentos          |

Cada reto tiene un objetivo claro. Tú decides cómo llegar.
